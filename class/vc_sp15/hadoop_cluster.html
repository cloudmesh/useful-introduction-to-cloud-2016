Introduction¶

Hadoop Cluster is a computational cluster especially for MapReduce data analysis applications. OpenStack on FutureSystems offers Hadoop Cluster with virtual instances so that large amounts of distributed processing data can be analyzed with better speed. Deploying Hadoop Cluster on OpenStack FutureSystems is offered by a manual and an automated instruction.

With Cloudmesh, Cloud Management Software, Hadoop Cluster can be deployed with a simple command execution. launcher command in Cloudmesh helps you deploy Hadoop Cluster with default settings automatically. You can deploy Hadoop Cluster with cm launcher by following this page: Deploying Hadoop Cluster with cm launcher. This tutorial taks 15-20 minutes. For more information of launcher, see here

If you are willing to configure Hadoop Cluster manually, please follow the page: Deploying Hadoop Cluster on India OpenStack. This tutorial takes 30-45 minutes.
